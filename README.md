História do jogo: A história do jogo é uma personagem principal que esta no meio de uma floresta pequena, perto de um cartelo, onde tem portais que aparecem monstros desses portais, 
e a personagem principal tem q impedir que os monstros entrem no castelo, dessa forma ela tem que atacar os montros.

Descrição do jogo: O jogo consiste em um tipo de tower defense, onde temos que proteger o castelo dos monstros.

Controles: Os principais controles do jogo são esses abaixo
w -> andar para frente
A -> andar para a esquerda
s -> andar para trás
D -> andar para a direita
Espaço -> pular
Shift -> desliza levemente no chão
Botão esquerdo do mouse -> ataca os monstros
